# 🩸 AI Blood Cell Diagnosis System

An end-to-end **Explainable AI system** that analyzes blood cell data to detect abnormalities, predict diseases, and provide human-readable medical explanations.

---

## 🚀 Features

* 🔍 **Anomaly Detection**

  * Classifies cells as Normal or Abnormal
  * Provides confidence scores

* 🧬 **Disease Prediction**

  * Predicts conditions like Leukemia, Anemia, Infection

* 🧠 **Explainable AI (SHAP)**

  * Shows why a prediction was made
  * Feature-level contribution analysis

* 🏥 **Human-Readable Medical Explanation**

  * Converts model output into clinical reasoning

* 📊 **Interactive UI (Streamlit)**

  * Upload CSV
  * Select cell
  * Get instant diagnosis

---

## 🧠 Tech Stack

* Python
* Scikit-learn (Random Forest)
* SHAP (Explainable AI)
* Streamlit (UI)
* Pandas, NumPy

## ⚙️ Setup Instructions

```bash

```


